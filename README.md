# Casa en Calma

Casa en Calma es una aplicación pensada para ayudar a organizar las rutinas del hogar, la limpieza, el bienestar personal y los recordatorios familiares de una manera sencilla, visual y motivadora.

## Objetivo del proyecto

El objetivo principal de esta aplicación es ayudar a crear hábitos diarios y semanales para mantener el hogar más organizado, reducir el estrés y mejorar el bienestar familiar.

## Funciones previstas

- Gestión de tareas del hogar.
- Rutinas diarias, semanales y mensuales.
- Recordatorios para limpieza, bienestar y organización.
- Registro del estado de ánimo diario.
- Seguimiento de hábitos.
- Motivación mediante frases positivas.
- Actividades para realizar con niños.
- Posible sistema de notificaciones.
- Posible versión con base de datos.

## Tecnologías previstas

- Python
- CustomTkinter
- MySQL
- GitHub
- Markdown
